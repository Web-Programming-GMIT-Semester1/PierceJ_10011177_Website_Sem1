Ubuntu Install Guide

Before proceeding read table carefully

Ubuntu System Requirements

This table describes the MINIMUM system requirements for installing Ubuntu

Componenet 32-Bit Requirements 64-Bit Requirements
Processor 700-MHz 700-MHz (64-Bit capable)
RAM 512 Minimum (1GB Recommended, 3GB Maximum) 2GB+
Harddrive Capacity 5GB+ 8GB+
Screen VGA 1024x768 screen resolution VGA 1024x768 screen resolution
Peripheral CD/DVD drive or a USB port (for installation media) CD/DVD drive or a USB port (for installation media)
Internet Access Preferable for Updates Preferable for Updates

*Machines that are 10 or more years old (originally preloaded with "Windows ME" or "Windows 2000") that don't meet these guideline will probably require some work to revive
(the RAM usually needs to be upgraded to the level described above). You could try a lighter-weight distro or a minimal install of Ubuntu.

Ubuntu Installation Guide

This guide describes the installtion for a clean installation of Ubuntu Desktop. IT does NOT cover Dual Booting Option. IT is intended for older laptops that cannot run latest windows effectively.

Step 1

Download the ISO file Depending on internet connection this could take over 1 hour.


Step 2
